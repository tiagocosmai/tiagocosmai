# Idiomas (pt / en / es)

- **`src/data/content.json`** — textos longos: hero, expertise, histórico, projetos, contato, rodapé. Cada campo traduzível usa `{ "pt": "...", "en": "...", "es": "..." }`. Campos únicos (nome, URLs, `imageKey`, `href`) ficam como string.
- **`src/data/resources.json`** — rótulos de UI: navegação, formulário, `aria-label`, placeholders, mensagens de erro/sucesso.

As bandeiras do seletor vêm de **`src/components/Flags.tsx`**: `LANGUAGE_OPTIONS` liga cada idioma ao componente SVG (`BRFlag`, `USFlag`, `ESFlag`). O menu na barra é um dropdown customizado (SVG não cabe em `<option>`).

O idioma escolhido fica em `localStorage` (`portfolio-locale`). A detecção inicial usa o idioma do navegador (`pt*` → PT, `es*` → ES, caso contrário EN).

Para novas chaves em `resources.json`, use sempre as três línguas e chame `t("chave")` no componente (dentro de `useLocale()`).
